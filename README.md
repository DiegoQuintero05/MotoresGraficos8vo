## 1er Parcial --- Sistema de Inputs para manipulacion de la posicion de un objeto.  
## Inputs:
### Movimiento
+ #### Tecla W -> Arriba.
+ #### Tecla S -> Abajo.
+ #### Tecla A -> Izquierda.
+ #### Tecla D -> Derecha.
+ #### Tecla R -> Profundidad contrario a la pantalla (Atras).
+ #### Tecla F -> Profundidad hacia la pantalla (Adelante).
### Cambio de Color
+ #### Num 1 ->  Color Inicial
+ #### Num 2 ->  Rojo
+ #### Num 3 ->  Naranja
+ #### Num 4 ->  Verde
